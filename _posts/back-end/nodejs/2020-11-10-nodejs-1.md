# Node.js란

---

- 2008년 크롬이 생기고 난 후, web뿐만 아니라 pc를 control하기 위한 js
- 요청이 들어오면 동적으로 생성
- [https://nodejs.org/en/download/](https://nodejs.org/en/download/)에서 다운로드

# 터미널에서 node.js 사용해보기

---

```
$ node // node.js 실행
```

- 계산 시키기

```
  > 3 + 3
  6
```

- 변수 선언도 가능

```
  > a = 10
  > a
  10
```

- javascript file 실행하기

```
  > node // 파일명.js
```
