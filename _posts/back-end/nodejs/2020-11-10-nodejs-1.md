<hr>
- 2008년 크롬이 생기고 난 후, web뿐만 아니라 pc를 control하기 위한 js
- 요청이 들어오면 동적으로 생성

# # 터미널에서 node.js 사용해보기
<hr>
<pre>
node <b>(node.js 실행)</b>
</pre>
- 계산 시키기
<pre>
> 3 + 3
<b>6</b>
</pre>
- 변수 선언도 가능
<pre>
> a = 10
> a
<b>10</b>
</pre>
- javascript file 실행하기
<pre>
> node <b>파일명</b>.js
</pre>
