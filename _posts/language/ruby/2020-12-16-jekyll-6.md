# # \_Draft 폴더 만들기

<hr>
- 프로젝트 폴더 바로 아래에 \_draft 폴더 생성
- 마크다운으로 포스트 작성
  <pre>
  $ jekyll run serve
  </pre>
  <b>포스트를 볼 수 없음</b>
  <pre>
  $ jekyll run serve --draft
  </pre>
- 날짜는 현재 날짜로 자동설정됨
